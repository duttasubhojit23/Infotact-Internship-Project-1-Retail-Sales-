Omnichannel Retail Sales & Inventory Analytics Dashboard
This project focuses on analyzing retail sales data collected from multiple locations and sales channels. By processing and exploring the data, we aim to uncover key sales trends, evaluate product performance, and understand customer purchasing patterns. The project builds a complete data analytics pipeline and presents the results through an interactive dashboard to support data-driven business decisions.

Excel Insights:-
Performed initial data cleaning and preprocessing using Excel Worked on data cleaning by handling missing values and removing duplicates and standardized date and numerical formats for consistency. Calculated column for revenue using quantity and unit price. Used sorting and filtering to validate data accuracy and built pivot tables to analyze sales by category, city, and product for further analysis. SQL Insights:-
Analyzed sales data using SQL to identify top-performing categories, cities, and products contributing most to overall revenue. Extracted time-based trends and customer behavior insights to support data-driven decisions for marketing, inventory, and sales strategy. Python EDA:-
Dataset imported from MySQL and analyzed using Pandas. Performed data cleaning and ensured correct data types for date and numerical columns. Created new features such as month and hour for time-based analysis and analyzed sales trends across months and identified seasonal patterns. Identified peak sales hours using hourly order analysis and evaluated product performance based on total revenue and quantity sold. Analyzed customer segments to understand purchasing behavior. Generated aggregated datasets for dashboard development.

Developed an interactive dashboard to visualize sales performance and trends. Implemented KPI section to track total revenue, total orders, and average order value. Visualized monthly and daily sales trends to identify performance patterns and analyzed peak sales hours to understand customer activity. Analyzed customer segments to understand contribution to total revenue. Identified top-performing products based on revenue andCompared sales performance across different cities. Conclusion
This project demonstrates a complete data analytics workflow from data cleaning to visualization using Excel, SQL, Python, and Power BI.
